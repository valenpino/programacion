BLOQUE 01

¿Qué es programar?

📖 Recuerda
INSTRUCCIÓN
Una orden concreta y exacta que le das a la máquina. La sigue tal cual — sin inventarse nada.
Ej: "Avanza 3m · gira 90° · deposita la caja"
CÓDIGO
El idioma de las máquinas. Igual que tú hablas español, las máquinas entienden código — bloques, Python, JavaScript...
Ej: los bloques de MakeCode son código visual
PROGRAMA
Lo que lee la máquina para saber qué tiene que hacer. Sin programa, la máquina no hace nada.
Ej: la app del móvil, el programa del semáforo, el GPS
FUNCIÓN
Cada máquina tiene sus propios HAZ — y tiene muchos. El semáforo sabe encender rojo, verde y ámbar. El robot sabe mover el brazo, girar la pinza, avanzar... Cada uno de esos es una función.
Robot → mover brazo · girar pinza · avanzar  ·  Semáforo → encender verde · encender rojo · encender ámbar

BLOQUE 02

Las 3 estructuras: HAZ · SI · REPITE

📖 Recuerda
HAZ
= Acción
Una instrucción directa. Se ejecuta siempre, en orden.
Pon la luz en rojo
Espera 30 segundos
SI
= Condicional
Hay una condición. Lo que hace depende de si se cumple o no.
¿Hay caja? →
cógela · si no, avanza
REPITE
= Bucle
Repite las instrucciones sin parar, una y otra vez.
El ciclo verde → ámbar → rojo
sin parar

🎯 Escenario: puerta automática del súper

La puerta del supermercado se abre sola cuando alguien se acerca. Funciona todo el día, sin apagarse. Tiene un sensor que detecta si hay alguien delante.

PARTE 1 — La puerta nunca se apaga
¿Qué estructura hace que la puerta esté siempre activa, comprobando si alguien se acerca?
PARTE 2 — Detectar si alguien se acerca
El sensor detecta a alguien → abre la puerta. De lo contrario → mantén cerrada. ¿Qué estructura es esta?
PARTE 3 — Abrir la puerta
"Activa el motor y desliza la puerta" — es la instrucción concreta del dispositivo. ¿Qué tipo de instrucción es?
PARTE 4 — Detectar un incendio
El programador añade una mejora: si el sensor de humo detecta fuego, la puerta se abre automáticamente para evacuar. ¿Qué estructura es esta nueva función?
